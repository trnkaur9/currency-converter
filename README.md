# 💱 Currency Converter

A minimalist, responsive web app for real-time currency conversion. [Live Demo](https://currency-converter-iwkg.onrender.com/)

---

## ✨ Features

- 🕒 **Live Rates:** Real-time accurate exchange data.
- 🌍 **Global Support:** Supports major world currencies.
- 🔄 **Quick Swap:** Reverse currencies with a single click.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)

---

## 🚀 Getting Started

### Quick Setup

1. **Clone the repo:**

```bash
git clone https://github.com/trnkaur9/currency-converter.git

cd ./currency-converter
```

3. **Install dependencies:**

```bash
npm i
```

3. **Run the app:**

```bash
node app.js
```

---

## 📖 How to Use

1. 💵 **Enter** the amount.
2. 🏳️ **Select** the "From" and "To" currencies.
3. 📊 **View** the instantly converted result.
